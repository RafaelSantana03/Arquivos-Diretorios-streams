# Arquivos, Diretórios e Streams em C#

Este repositório contém anotações e exercícios práticos sobre manipulação de arquivos, diretórios e streams em **C#** e **.NET**.  
O conteúdo foi desenvolvido como parte dos estudos da linguagem, explorando recursos da biblioteca padrão para entrada/saída de dados.

## 📂 Conteúdo

Os estudos foram baseados na **Seção 10: Arquivos, Diretórios e Streams** do curso **C# Essencial** do Macoratti, cobrindo os seguintes tópicos:

- Classe **File**
- Métodos e operações adicionais com **File**
- Classe **FileInfo**
- Classes **Directory** e **DirectoryInfo**
- Operações avançadas com **DirectoryInfo**
- Classe **Path**
- Streams: **FileStream**, **StreamReader** e **StreamWriter**
- Exercícios com **FileStream** e **StreamReader**
- Streams com bloco **using**
- Exercícios práticos de **Streams**

## 📝 Exercícios

O repositório inclui **15 exercícios** para prática, como:

1. Verificar se um arquivo existe e exibir sua data de criação.  
2. Copiar um arquivo de um diretório para outro.  
3. Criar e escrever informações em um arquivo.  
4. Criptografar e salvar um arquivo usando AES.  
5. Converter imagem para texto codificado Base64.  
... e muito mais.
